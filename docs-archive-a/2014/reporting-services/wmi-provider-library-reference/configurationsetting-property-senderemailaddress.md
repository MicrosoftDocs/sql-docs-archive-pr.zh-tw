# <a name="senderemailaddress-property-wmi-msreportserver_configurationsetting"></a>SenderEmailAddress 屬性 (WMI MSReportServer_ConfigurationSetting)
  取得用以從報表伺服器傳送電子郵件的位址。 唯讀。  
  
## <a name="syntax"></a>語法  
  
```vb  
Public Dim SenderEmailAddress As String  
```  
  
```csharp  
public string SenderEmailAddress;  
```  
  
## <a name="property-values"></a>屬性值  
 唯讀 `String` 物件，代表報表伺服器所使用的電子郵件地址。
