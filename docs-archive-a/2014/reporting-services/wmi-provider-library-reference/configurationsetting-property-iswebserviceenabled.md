# <a name="iswebserviceenabled-property-wmi-msreportserver_configurationsetting"></a>IsWebServiceEnabled 屬性 (WMI MSReportServer_ConfigurationSetting)
  指出報表伺服器 Web 服務是否已啟用。 唯讀。  
  
## <a name="syntax"></a>語法  
  
```vb  
Public Dim IsWebServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWebServiceEnabled;  
```  
  
## <a name="property-values"></a>屬性值  
 唯讀的 `Boolean` 值。 值為 `true` 表示報表伺服器 Web 服務已啟用。
