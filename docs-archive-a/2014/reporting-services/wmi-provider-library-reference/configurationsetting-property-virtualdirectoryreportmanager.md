# <a name="virtualdirectoryreportmanager-property-wmi-msreportserver_configurationsetting"></a>VirtualDirectoryReportManager 屬性 (WMI MSReportServer_ConfigurationSetting)
  傳回報表管理員應用程式的虛擬目錄。 唯讀。  
  
## <a name="syntax"></a>語法  
  
```vb  
Public Dim VirtualDirectoryReportManager As String  
```  
  
```csharp  
public string VirtualDirectoryReportManager;  
```  
  
## <a name="property-values"></a>屬性值  
 包含虛擬目錄的 `string`。
