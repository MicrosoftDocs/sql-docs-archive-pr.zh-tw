# <a name="select-the-report-type"></a>選取報表類型
  使用報表精靈的這個頁面，即可選取表格式報表或矩陣報表。  
  
## <a name="options"></a>選項  
 **表格式**  
 建立包含資料表的報表。 資料表是將資料排列成資料行和資料列的資料區域。 資料表有一組靜態的資料行，至於資料表中的資料列數目則是依資料集內的資料而定。  
  
 精靈中的後續頁面會提供定義資料表的選項。  
  
 **Matrix**  
 建立包含矩陣的報表。 矩陣是將資料排列成資料行和資料列的資料區域。 矩陣提供與交叉資料表和樞紐資料表類似的功能。 資料表有一組靜態的資料行，但矩陣不同，它可以有動態的資料行。  
  
 精靈中的後續頁面會提供定義矩陣的選項。
