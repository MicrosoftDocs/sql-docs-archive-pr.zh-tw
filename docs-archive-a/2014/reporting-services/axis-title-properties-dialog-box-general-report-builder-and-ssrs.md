# <a name="axis-title-properties-dialog-box-general-report-builder-and-ssrs"></a>軸標題屬性對話方塊、一般 (報表產生器及 SSRS)
  選取 [軸標題屬性]**** 對話方塊上的 [一般]**** 來定義軸標題。 軸標題會顯示在 X 軸之下，或 Y 軸旁邊。  
  
## <a name="options"></a>選項  
 **標題**  
 輸入標題或代表軸標題的運算式。  
  
 **標題對齊**  
 選取對齊方式，或輸入代表軸標題相對於軸交集之對齊的運算式。  
  
-   **接近** ：朝兩個軸交集的地方顯示軸標題。  
  
-   **置中** ：在軸的中央顯示軸標題。  
  
-   **遠方** ：遠離兩個軸交集的地方顯示軸標題。
