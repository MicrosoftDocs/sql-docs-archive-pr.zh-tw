# <a name="enter-data-source-credentials-dialog-box"></a>輸入資料來源認證對話方塊
  使用 [**輸入資料來源認證**] 對話方塊來提供您的 [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] 認證，讓您可以從資料來源中取出資料。  
  
## <a name="options"></a>選項。  
 **使用者名稱**  
 輸入使用者名稱，即可登入資料來源。  
  
 **密碼**  
 輸入密碼，即可登入資料來源。
