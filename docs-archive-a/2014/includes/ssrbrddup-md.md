您可以在報表產生器以及 SQL Server Data Tools 的報表設計師中建立和修改報表定義 \(.rdl\)。 每種撰寫環境都會提供不同的方式來建立、開啟和儲存報表與相關的項目。 如需詳細資訊，請參閱 microsoft.com 網站上的**在報表設計師及報表產生器中設計報表\(SSRS\)**。
