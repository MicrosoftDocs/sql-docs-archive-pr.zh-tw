# <a name="symmetric-keys-on-system-databases"></a>系統資料庫上的對稱金鑰
  此規則會檢查 master、msdb、model 和 tempdb 資料庫中是否有使用者建立的對稱金鑰。  
  
## <a name="best-practices-recommendations"></a>最佳做法建議  
 請勿在系統資料庫中建立對稱金鑰。
