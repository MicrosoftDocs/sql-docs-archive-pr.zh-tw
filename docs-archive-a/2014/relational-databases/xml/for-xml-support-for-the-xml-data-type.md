## <a name="returning-xml-from-a-user-defined-function"></a><span data-ttu-id="d31eb-116">從使用者自訂函數傳回 XML</span><span class="sxs-lookup"><span data-stu-id="d31eb-116">Returning XML from a User-defined Function</span></span>  
 <span data-ttu-id="d31eb-117">可利用 FOR XML 查詢，從傳回下列項目之一的使用者自訂函數，傳回 XML：</span><span class="sxs-lookup"><span data-stu-id="d31eb-117">FOR XML queries can be used for returning XML from a user-defined function that returns either of the following:</span></span>  
  
-   <span data-ttu-id="d31eb-118">具有單一 `xml` 類型資料行的資料表</span><span class="sxs-lookup"><span data-stu-id="d31eb-118">A table with a single `xml` type column</span></span>  
  
-   <span data-ttu-id="d31eb-119">`xml` 類型的執行個體</span><span class="sxs-lookup"><span data-stu-id="d31eb-119">An instance of the `xml` type</span></span>  
  
 <span data-ttu-id="d31eb-120">例如，下列使用者定義函數會傳回具有單一 `xm`l 類型資料行的資料表：</span><span class="sxs-lookup"><span data-stu-id="d31eb-120">For example, the following user-defined function returns a table with a single column of `xm`l type:</span></span>  
  
```  
USE AdventureWorks2012;  
GO  
CREATE FUNCTION dbo.MyUDF (@ProudctModelID int)  
RETURNS @T TABLE  
  (  
     ProductDescription xml  
  )  
AS  
BEGIN  
  INSERT @T  
     SELECT CatalogDescription.query('  
declare namespace PD="http://www.adventure-works.com/schemas/products/description";  
                    //PD:ProductDescription  ')  
     FROM Production.ProductModel  
     WHERE ProductModelID = @ProudctModelID  
  RETURN  
END;  
```  
  
 <span data-ttu-id="d31eb-121">您可以執行使用者定義函數，並查詢它所傳回的資料表。</span><span class="sxs-lookup"><span data-stu-id="d31eb-121">You can execute the user-defined function and query the table returned by it.</span></span> <span data-ttu-id="d31eb-122">在此範例中，會將透過查詢資料表而傳回的 XML 指派給 `xml` 類型的變數。</span><span class="sxs-lookup"><span data-stu-id="d31eb-122">In this example, the XML returned by querying the table is assigned to an `xml` type variable.</span></span>

```  
  
 <span data-ttu-id="d31eb-123">以下是另一個使用者定義函數範例。</span><span class="sxs-lookup"><span data-stu-id="d31eb-123">This is another example of a user-defined function.</span></span> <span data-ttu-id="d31eb-124">這個使用者定義函數會傳回 `xml` 類型的執行個體。</span><span class="sxs-lookup"><span data-stu-id="d31eb-124">This user-defined function returns an instance of the `xml` type.</span></span> <span data-ttu-id="d31eb-125">在此範例中，使用者定義函數會傳回 XML 類型的執行個體，因為有指定結構描述命名空間。</span><span class="sxs-lookup"><span data-stu-id="d31eb-125">In this example, the user-defined function returns a typed XML instance, because the schema namespace is specified.</span></span>

CREATE FUNCTION MyUDF (@ProductModelID int)

AS

  declare @x xml  
  set @x =   ( SELECT CatalogDescription  
          FROM Production.ProductModel  
          WHERE ProductModelID = @ProductModelID )  
  return @x  
END;  
```
