# <a name="mssql_repl-2147201021"></a>MSSQL_REPL-2147201021
    
## <a name="message-details"></a>訊息詳細資料  
  
|||  
|-|-|  
|產品名稱|SQL Server|  
|事件識別碼|-2147201021|  
|事件來源|MSSQLServer|  
|符號名稱||  
|訊息文字|發行集 '%1' 並非使用中。 若要啟動這個發行集，請啟動快照集代理程式來產生初始快照集。|  
  
## <a name="explanation"></a>說明  
 指定的發行集處於非使用中狀態。  
  
## <a name="user-action"></a>使用者動作  
 若要啟動發行集，請執行發行集的快照集代理程式。
