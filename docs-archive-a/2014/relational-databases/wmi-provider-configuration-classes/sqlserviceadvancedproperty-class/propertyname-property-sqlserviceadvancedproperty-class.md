# <a name="propertyname-property-sqlserviceadvancedproperty-class"></a>PropertyName 屬性 (SqlServiceAdvancedProperty 類別)
  取得或設定進階屬性的名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.PropertyName [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 代表進階屬性的 [SqlServiceAdvancedProperty 類別](sqlserviceadvancedproperty-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定進階屬性名稱的字串值。
