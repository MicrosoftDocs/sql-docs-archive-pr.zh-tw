# <a name="protocolname-property-servernetworkprotocolipaddress-class"></a>ProtocolName 屬性 (ServerNetworkProtocolIPAddress 類別)
  取得參考 IP 位址的伺服器網路通訊協定名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.ProtocolName [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 代表實例上網路通訊協定之 IP 位址的[ServerNetworkProtocolIPAdress 類別](servernetworkprotocolipaddress-class.md)物件 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定伺服器網路通訊協定名稱的字串值。
