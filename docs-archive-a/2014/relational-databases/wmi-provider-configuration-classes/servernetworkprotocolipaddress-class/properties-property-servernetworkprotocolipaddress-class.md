# <a name="properties-property-servernetworkprotocolipaddress-class"></a>Properties 屬性 (ServerNetworkProtocolIPAddress 類別)

## <a name="syntax"></a>語法  
  
```  
  
object  
.Properties [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 代表 [執行個體上網路通訊協定之 IP 位址的](servernetworkprotocolipaddress-class.md) ServerNetworkProtocolIPAdress 類別 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]物件。
