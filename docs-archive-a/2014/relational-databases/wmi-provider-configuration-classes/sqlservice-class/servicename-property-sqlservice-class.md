# <a name="servicename-property-sqlservice-class"></a>ServiceName 屬性 (SqlService 類別)
  取得服務的名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.ServiceName [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 表示此服務的 [SqlService 類別](sqlservice-class.md) 物件。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定此服務之名稱的字串值。
