# <a name="sqlservice-class"></a>SqlService 類別
  [SqlService 類別](sqlservice-class.md)代表服務的實例 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 。  
  
 [SqlService 類別](sqlservice-class.md)是用來以程式設計方式管理 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 服務。 使用[SqlService 類別](sqlservice-class.md)物件，您可以執行下列工作：  
  
-   連接到服務。  
  
-   修改用來連接服務的帳戶和密碼。  
  
-   啟動、停止、暫停和繼續服務。  
  
-   檢查服務的屬性。
