# <a name="protocoldisplayname-property-clientnetworkprotocol-class"></a>ProtocolDisplayName 屬性 (ClientNetworkProtocol 類別)
  取得[設定用戶端通訊協定](https://technet.microsoft.com/library/ms181035.aspx)所指定之用戶端網路通訊協定的顯示名稱。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.ProtocolDisplayName [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 代表用戶端所使用之網路通訊協定的[ClientNetworkProtocol 類別](clientnetworkprotocol-class.md)物件 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定用戶端網路通訊協定之顯示名稱的字串值。
