# <a name="numberofflags-property-clientnetworkprotocol-class"></a>NumberOfFlags 屬性 (ClientNetworkProtocol 類別)
  取得 SetOrderValue 方法所指定的用戶端網路通訊協定所需的旗標選項數目， [ (ClientNetworkProtocol 類別) ](clientnetworkprotocol-class.md)。  
  
## <a name="syntax"></a>語法  
  
```  
  
object  
.NumberofFlags [= value]  
```  
  
## <a name="parts"></a>組件  
 *object*  
 代表用戶端所使用之網路通訊協定的[ClientNetworkProtocol 類別](clientnetworkprotocol-class.md)物件 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 。  
  
## <a name="property-valuereturn-value"></a>屬性值/傳回值  
 指定 `Uint32` 屬性所參考之用戶端網路通訊協定所需旗標選項數的 `OrderValue` 值。
