# <a name="serversettingsgeneralflag-class"></a>ServerSettingsGeneralFlag 類別
  [ServerSettingsGeneralFlag 類別](serversettingsgeneralflag-class.md)代表實例上的一般旗標設定 [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] 。  
  
 此類別可用來進行以下作業：  
  
-   檢視旗標名稱。  
  
-   檢視和修改旗標值。
