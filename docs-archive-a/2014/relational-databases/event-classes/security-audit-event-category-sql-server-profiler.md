# <a name="security-audit-event-category-sql-server-profiler"></a>Security Audit 事件類別目錄 (SQL Server Profiler)

|主題|描述|

|[Audit App Role Change Password 事件類別](audit-app-role-change-password-event-class.md)|表示已變更應用程式角色的密碼。|
