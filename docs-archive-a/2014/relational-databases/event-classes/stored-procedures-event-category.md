# <a name="stored-procedures-event-category"></a>Stored Procedures 事件類別目錄
  **Stored Procedures** 事件類別目錄包含一般的預存程序事件。  
  
## <a name="in-this-section"></a>本節內容  
  
|主題|描述|  
|-----------|-----------------|  
|[RPC:Completed 事件類別](rpc-completed-event-class.md)|指出遠端程序呼叫 (RPC) 已完成。|  
|[PreConnect:Completed 事件類別](preconnect-completed-event-class.md)|指出資源管理員分類函數執行完成。|  
|[PreConnect:Starting 事件類別](preconnect-starting-event-class.md)|指出資源管理員分類函數開始執行。|  
|[RPC Output Parameter 事件類別](rpc-output-parameter-event-class.md)|追蹤遠端程序呼叫執行之後的輸出參數值。|  
|[RPC:Starting 事件類別](rpc-starting-event-class.md)|指出遠端程序呼叫已開始。|  
|[SP:CacheHit 事件類別](sp-cachehit-event-class.md)|指出預存程序已在快取中。|  
|[SP:CacheInsert 事件類別](sp-cacheinsert-event-class.md)|指出已將預存程序放入快取中。|  
|[SP:CacheMiss 事件類別](sp-cachemiss-event-class.md)|指出在快取中找不到預存程序。|  
|[SP:CacheRemove 事件類別](sp-cacheremove-event-class.md)|指出已從快取移除預存程序。|  
|[SP:Completed 事件類別](sp-completed-event-class.md)|指出已完成預存程序的執行。|  
|[SP:Recompile 事件類別](sp-recompile-event-class.md)|指出已重新編譯預存程序。|  
|[SP:Starting 事件類別](sp-starting-event-class.md)|指出預存程序的執行已開始。|  
|[SP:StmtCompleted 事件類別](sp-stmtcompleted-event-class.md)|指出已經完成預存程序中的 [!INCLUDE[tsql](../../includes/tsql-md.md)] 陳述式。|  
|[SP:StmtStarting 事件類別](sp-stmtstarting-event-class.md)|指出已經開始預存程序中的 [!INCLUDE[tsql](../../includes/tsql-md.md)] 陳述式。|
