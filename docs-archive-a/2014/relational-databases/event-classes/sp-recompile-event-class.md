# <a name="sprecompile-event-class"></a>SP:Recompile 事件類別
  SP:Recompile 事件類別表示有某個預存程序、觸發程序或使用者自訂函數已重新編譯。 這個事件類別報告的重新編譯會在陳述式層級發生。  
  
 若要追蹤陳述式層級的重新編譯，建議使用 SQL:StmtRecompile 事件類別。 SP:Recompile 事件類別已被取代。 如需詳細資訊，請參閱 [SQL:StmtRecompile 事件類別](sql-stmtrecompile-event-class.md)。
