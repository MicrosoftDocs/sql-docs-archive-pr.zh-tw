# <a name="browse-for-objects-object-explorer"></a>瀏覽物件 (物件總管)
  顯示所選取類型的物件清單。 透過選取對應的核取方塊，從這份清單中選取一或多個物件。 [瀏覽物件]  方塊只會顯示適用於該動作的項目，並只顯示您有權限檢視的項目。  
  
## <a name="options"></a>選項。  
 **核取方塊**  
 選取以包含物件。  
  
 **圖示**  
 代表物件類型的圖示。  
  
 **名稱**  
 物件的名稱。  
  
 **型別**  
 資料庫物件的類型。
