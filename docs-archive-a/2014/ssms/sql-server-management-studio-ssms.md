# <a name="sql-server-management-studio"></a>SQL Server Management Studio
   是用於管理 SQL Server 基礎結構的整合式環境。 [!INCLUDE[ssManStudio](../includes/ssmanstudio-md.md)] 提供工具來設定、監視以及管理 SQL Server 執行個體。 也提供工具來部署、監視以及升級資料層元件 (例如應用程式所使用的資料庫和資料倉儲)，以及建立查詢和指令碼。  
  
## <a name="benefits-of-sql-server-management-studio"></a>SQL Server Management Studio 的優點  
  是用於管理 SQL Server 物件的綜合性公用程式，結合簡單易用的圖形介面以及非常豐富的指令碼功能。  可用來管理 、、 及 。
