# <a name="other-ways-of-viewing-the-code-window"></a>程式碼視窗的其他檢視方式
  您可以利用多種方式，將 [程式碼] 視窗設定成會顯示多個程式碼視窗。  
  
## <a name="using-multiple-code-windows"></a>使用多個程式碼視窗  
  
#### <a name="to-view-and-manipulate-multiple-code-windows-at-once"></a>若要同時檢視和操作多個程式碼視窗  
  
1.  在 [SQL 編輯器]**** 工具列上，按一下 [新增查詢]**** 來開啟第二個 [查詢編輯器] 視窗。  
  
2.  若要同時看到兩個程式碼視窗，請以滑鼠右鍵按一下 [查詢編輯器] 的標題列，然後選取 [新增水平索引標籤群組]****。 此時兩個查詢視窗會以水平窗格出現。  
  
3.  按一下上方的 [查詢編輯器] 視窗，使其變成使用中狀態，然後按一下 [新增查詢]**** 開啟第三個查詢視窗。 它會顯示成上方視窗中的索引標籤。  
  
4.  在 [視窗]**** 功能表上，按一下 [移至下一個索引標籤群組]****。 此時第三個視窗會移至下面的索引標籤群組中。 您可以利用這些選項，以各種不同的方式來設定視窗。  
  
5.  關閉第二和第三個查詢視窗。  
  
> [!NOTE]  
>  若要檢視查詢編輯器的全螢幕模式，請按一下 [查詢編輯器] 視窗中的任何位置，再按 SHIFT+ALT+ENTER。
