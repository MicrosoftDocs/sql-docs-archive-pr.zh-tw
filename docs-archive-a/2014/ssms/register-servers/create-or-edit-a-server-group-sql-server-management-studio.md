# <a name="create-or-edit-a-server-group-sql-server-management-studio"></a><span data-ttu-id="10a9d-102">建立或編輯伺服器群組 (SQL Server Management Studio)</span><span class="sxs-lookup"><span data-stu-id="10a9d-102">Create or Edit a Server Group (SQL Server Management Studio)</span></span>
  <span data-ttu-id="10a9d-103">本主題描述如何透過建立伺服器群組，然後將伺服器放入伺服器群組中的方式，組織 [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] 中 [已註冊的伺服器] 內的伺服器。</span><span class="sxs-lookup"><span data-stu-id="10a9d-103">This topic describes how to organize the servers in Registered Servers in [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] by creating server groups, and placing the servers in the server groups.</span></span> <span data-ttu-id="10a9d-104">您隨時都可在「已註冊的伺服器」中建立伺服器群組，您也可以在註冊伺服器時建立伺服器群組。</span><span class="sxs-lookup"><span data-stu-id="10a9d-104">You can create server groups in Registered Servers at any time, or you can create server groups when you register servers.</span></span>  
  
##  <a name="SSMSProcedure"></a>  
  
#### <a name="to-create-a-server-group-in-registered-servers"></a><span data-ttu-id="10a9d-105">若要在已註冊的伺服器中建立伺服器群組</span><span class="sxs-lookup"><span data-stu-id="10a9d-105">To create a server group in Registered Servers</span></span>  
  
1.  <span data-ttu-id="10a9d-106">在 [已註冊的伺服器] 中，按一下 [已註冊的伺服器] 工具列上的伺服器類型。</span><span class="sxs-lookup"><span data-stu-id="10a9d-106">In Registered Servers, click the server type on the Registered Servers toolbar.</span></span> <span data-ttu-id="10a9d-107">如果看不見「已註冊的伺服器」，請在 **[檢視]** 功能表中按一下 **[已註冊的伺服器]** 。</span><span class="sxs-lookup"><span data-stu-id="10a9d-107">If Registered Servers is not visible, click **Registered Servers** on the **View** menu.</span></span>  
  
2.  <span data-ttu-id="10a9d-108">以滑鼠右鍵按一下伺服器或伺服器群組，並指向 [新增]  ，然後按一下 [伺服器群組]  。</span><span class="sxs-lookup"><span data-stu-id="10a9d-108">Right-click a server or a server group, point to **New**, and then click **Server Group**.</span></span>  
  
3.  <span data-ttu-id="10a9d-109">在 **[新增伺服器群組]** 對話方塊的 **[群組名稱]** 清單方塊中，輸入伺服器群組的唯一名稱。</span><span class="sxs-lookup"><span data-stu-id="10a9d-109">In the **New Server Group** dialog box, in the **Group name** list box, type a unique name for the server group.</span></span> <span data-ttu-id="10a9d-110">在「已註冊的伺服器」樹狀結構的目前位置上，伺服器群組名稱必須是唯一的。</span><span class="sxs-lookup"><span data-stu-id="10a9d-110">The server group name must be unique for the current location in the Registered Servers tree.</span></span>  
  
4.  <span data-ttu-id="10a9d-111">(選擇性) 在 **[群組描述]** 清單方塊中，輸入用來描述伺服器群組的易記名稱，例如「拉丁美洲的財務伺服器」。</span><span class="sxs-lookup"><span data-stu-id="10a9d-111">In the **Group description** list box, optionally type a friendly name that describes the server group, for example, "Finance Servers for Latin America."</span></span>  
  
5.  <span data-ttu-id="10a9d-112">在 **[選取新伺服器群組的位置]** 方塊中，按一下群組的位置，然後按一下 **[儲存]** 。</span><span class="sxs-lookup"><span data-stu-id="10a9d-112">In the **Select a location for the new server group** box, click a location for the group, and then click **Save**.</span></span>  
  
    > [!NOTE]  
    >  <span data-ttu-id="10a9d-113">在註冊伺服器時，您也可以按一下 [新增群組]  ，然後完成 [新增群組]  對話方塊來建立新伺服器群組。</span><span class="sxs-lookup"><span data-stu-id="10a9d-113">You can also create a new server group while you are registering a server by clicking **New Group**, and then completing the **New Group** dialog box.</span></span>
