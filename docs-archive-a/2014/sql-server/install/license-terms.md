# <a name="license-terms"></a>授權條款
  使用 [授權條款] 頁面，即可接受這個版本的授權條款。  
  
## <a name="options"></a>選項  
 若要繼續進行安裝，您必須接受 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 授權條款。  
  
 使用 [**列印**] 按鈕來列印此版本的授權條款複本。 若要結束安裝，請按一下 [**取消**]。
