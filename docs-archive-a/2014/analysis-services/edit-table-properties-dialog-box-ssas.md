# <a name="edit-table-properties-dialog-box-ssas"></a>編輯資料表屬性對話方塊 (SSAS)
  **[編輯資料表屬性]** 對話方塊可讓您檢視與修改使用 [資料表匯入精靈] 匯入至模型設計師之資料表的屬性。 若要存取此對話方塊，請在模型設計師中選取資料表，然後按一下 **[資料表]** 功能表，再按一下 **[資料表屬性]**。  
  
## <a name="ui-element-list"></a>UI 元素清單  
 依您一開始是透過從清單中選取資料表或使用 SQL 查詢匯入資料，此對話方塊的選項都不同。  
  
## <a name="table-preview-mode"></a>資料表預覽模式  
 **資料表名稱**  
 顯示模型中資料表的名稱。  
  
> [!NOTE]  
>  您無法在此處編輯名稱。 不過，您可以在模型設計師底部的資料表索引標籤上按一下滑鼠右鍵，來變更資料表的名稱。  
  
 **連線名稱**  
 顯示目前正在使用之連接的名稱。  
  
 **來源名稱**  
 顯示或變更從中取得資料的資料表。  
  
 如果您將來源變更為具有與目前資料表不同之資料行的資料表，就會顯示資料行不同的警告訊息。 然後，您必須選取要放入目前資料表中的資料行，並按一下 **[儲存]**。 您可以選取資料表左邊的核取方塊，以取代整個資料表。  
  
> [!NOTE]  
>  當您變更資料表的資料來源時，基本上是以新來源資料表的內容取代目前資料表的內容。  
  
 **資料行名稱來自**  
 |||  
|-|-|  
|**Source**|選取此選項，以所選來源資料表的資料行名稱取代目前的資料行名稱。|  
|**型號**|選取此選項，以使用目前資料行在模型中顯示的名稱。|  
  
 **重新整理預覽**  
 按一下以檢視目前所選來源資料表中的資料行。  
  
 **切換到**  
 |||  
|-|-|  
|**資料表預覽**|選取此選項，以預覽所選資料表及有限數目的資料列。|  
|**查詢編輯器**|選取此選項，以檢視對所選資料來源執行的查詢。 並非所有的資料來源都可使用這個選項。|  
  
 **資料行標頭中的核取方塊**  
 選取核取方塊可將資料行納入資料匯入作業。 清除核取方塊可從資料匯入作業排除資料行。  
  
 **資料行標頭中的向下箭號按鈕**  
 篩選資料行中的資料。  
  
 **清除資料列篩選**  
 按一下以移除任何已套用的篩選。  
  
 **確定**  
 按一下以套用所做的全部變更，包括取代資料行。  
  
## <a name="query-design-mode"></a>查詢設計模式  
 **資料表名稱**  
 顯示模型中資料表的名稱。  
  
> [!NOTE]  
>  您無法在此編輯名稱，但是可以滑鼠右鍵按一下設計師底部的資料表索引標籤來變更資料表的名稱。  
  
 **連線名稱**  
 顯示目前正在使用之連接的名稱。  
  
 **切換到**  
 |||  
|-|-|  
|**資料表預覽**|選取此選項，以預覽所選資料表及少數資料列。|  
|**查詢編輯器**|選取此選項，以檢視將對所選資料來源發出的查詢。|  
  
 **Sql 語句**  
 顯示依目前資料來源發出以擷取資料列的 SQL 陳述式。 在預設情況下會擷取所有資料列，但是您可以設計篩選器或以手動方式編輯 SQL 陳述式，只擷取資料列的子集。  
  
 **驗證**  
 按一下以針對所選資料來源和提供者，驗證陳述式的語法是否正確。  
  
 **設計**  
 按一下以開啟視覺化查詢設計工具，並建立查詢陳述式。 如需有關如何使用設計工具的詳細資訊，請從設計工具按 F1。  
  
 **確定**  
 按一下以套用所做的全部變更，包括取代資料行。
