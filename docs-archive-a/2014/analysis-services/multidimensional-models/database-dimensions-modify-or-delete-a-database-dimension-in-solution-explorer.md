# <a name="modify-or-delete-a-database-dimension-in-solution-explorer"></a>在方案總管中修改或刪除資料庫維度
  您可以在 [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)]中使用維度設計師修改或刪除維度。 修改維度時，必須等到您處理維度之後，使用者才可以使用變更後的維度。 如需詳細資訊，請參閱多[維度模型物件處理](processing-a-multidimensional-model-analysis-services.md)。  
  
### <a name="to-modify-a-dimension-in-sql-server-data-tools"></a>若要在 SQL Server 資料工具中修改維度  
  
1.  在方案總管中，以滑鼠右鍵按一下您要修改的維度，然後按一下 [開啟]****。  
  
2.  在 [維度結構]**** 索引標籤的 [屬性]****、[階層]**** 或 [資料來源檢視]**** 窗格中，按一下想要修改的項目，然後進行變更。  
  
     如需您可以進行之變更類型的詳細資訊，請參閱多[維度模型物件處理](processing-a-multidimensional-model-analysis-services.md)。  
  
### <a name="to-delete-a-dimension-in-sql-server-data-tools"></a>若要在 SQL Server 資料工具中刪除維度  
  
-   在方案總管中，以滑鼠右鍵按一下您要刪除的維度，然後按一下 [刪除]****。
