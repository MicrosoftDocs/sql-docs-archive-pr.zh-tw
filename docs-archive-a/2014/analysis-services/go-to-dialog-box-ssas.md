# <a name="go-to-dialog-box-ssas"></a>移至對話方塊 (SSAS)
  **[移至]** 對話方塊可讓您跳到資料表中的特定資料行。  
  
## <a name="ui-element-list"></a>UI 元素清單  
  
|||  
|-|-|  
|詞彙|定義|  
|**轉到 (G) **|從 **[移至]** 清單中選取資料行，然後按一下 **[確定]**。|
