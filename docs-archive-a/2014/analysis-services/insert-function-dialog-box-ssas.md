# <a name="insert-function-dialog-box-ssas"></a>插入函數對話方塊 (SSAS)
  **[插入函數]** 對話方塊可讓您從函數清單選擇建立公式時可以使用的函數。 若要從模型設計師存取此對話方塊，在每個資料表上方的公式列中，按一下函數 (**fx**) 按鈕。 如需有關選擇於公式中使用之函數的詳細資訊，請參閱＜DAX 簡介＞和＜建立公式＞。  
  
|||  
|-|-|  
|項目|描述|  
|**選取類別**|如果您約略知道所需要的函數種類，請從清單選擇一個類別目錄，或者選取 **[全部]** 來檢視依字母順序排列的函數清單。|  
|**選取函式**|顯示所選類別目錄中的函數清單。|  
|**說明**|顯示函數用途的描述，以及任何必要或選用的引數，例如資料行名稱和運算式。|  
  
## <a name="function-categories"></a>函數類別目錄  
 Data Analysis Expressions (DAX) 會在 [插入函數]**** 對話方塊中提供下列類型的函數類別目錄。  
  
 全部  
  
 日期和時間  
  
 Filter  
  
 邏輯  
  
 數學與三角函數  
  
 統計  
  
 Text  
  
## <a name="measures-and-formulas"></a>量值和公式  
 只有在您建立公式時，才可以使用 **[插入函數]** 對話方塊。 您可以在計算結果欄、樞紐分析表或樞紐分析圖中建立計算。 您針對在樞紐分析表中使用而明確建立的公式也稱為 *「量值」*(Measure)。 如需詳細資訊，請參閱[建立導出資料行 &#40;SSAS 表格式&#41;](tabular-models/ssas-calculated-columns-create-a-calculated-column.md) 和[建立及管理量值 &#40;SSAS 表格式&#41;](tabular-models/measures-ssas-tabular.md)。
