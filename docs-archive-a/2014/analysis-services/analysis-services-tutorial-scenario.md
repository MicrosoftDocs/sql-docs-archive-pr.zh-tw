## <a name="current-analysis-environment"></a><span data-ttu-id="d1309-112">目前分析環境</span><span class="sxs-lookup"><span data-stu-id="d1309-112">Current Analysis Environment</span></span>  
 <span data-ttu-id="d1309-113">為了支援銷售和行銷團隊以及資深管理的資料分析需求，該公司目前是從 [!INCLUDE[ssSampleDBnormal](../includes/sssampledbnormal-md.md)] 資料庫取得交易資料，再從試算表取得非交易資訊，如銷售配額，然後將這項資訊合併到 **AdventureWorksDW2012** 關聯式資料倉儲。</span><span class="sxs-lookup"><span data-stu-id="d1309-113">To support the data analysis needs of the sales and marketing teams and of senior management, the company currently takes transactional data from the [!INCLUDE[ssSampleDBnormal](../includes/sssampledbnormal-md.md)] database, and non-transactional information such as sales quotas from spreadsheets, and consolidates this information into the **AdventureWorksDW2012** relational data warehouse.</span></span> <span data-ttu-id="d1309-114">不過，關聯式資料倉儲帶來下列挑戰：</span><span class="sxs-lookup"><span data-stu-id="d1309-114">However, the relational data warehouse presents the following challenges:</span></span>  
  
-   <span data-ttu-id="d1309-115">報表是靜態的。</span><span class="sxs-lookup"><span data-stu-id="d1309-115">Reports are static.</span></span> <span data-ttu-id="d1309-116">使用者無法互動地瀏覽報表中的資料來取得更詳細的資訊，如同他們在處理 [!INCLUDE[msCoName](../includes/msconame-md.md)] Office Excel 樞紐資料表一般。</span><span class="sxs-lookup"><span data-stu-id="d1309-116">Users have no way to interactively explore the data in the reports to obtain more detailed information, such as they could do with a [!INCLUDE[msCoName](../includes/msconame-md.md)] Office Excel pivot table.</span></span> <span data-ttu-id="d1309-117">雖然現有的預先定義報表集對許多使用者而言已經足夠，但更進階的使用者需要直接查詢來存取資料庫中的互動式查詢和特殊報表。</span><span class="sxs-lookup"><span data-stu-id="d1309-117">Although the existing set of predefined reports is sufficient for many users, more advanced users need direct query access to the database for interactive queries and specialized reports.</span></span> <span data-ttu-id="d1309-118">不過，因為**AdventureWorksDW2012**資料庫的複雜性，所以這類使用者需要太多時間來學習如何建立有效的查詢。</span><span class="sxs-lookup"><span data-stu-id="d1309-118">However, because of the complexity of the **AdventureWorksDW2012** database, too much time is needed for such users to learn how to create effective queries.</span></span>  
  
-   <span data-ttu-id="d1309-119">查詢效能變化很大。</span><span class="sxs-lookup"><span data-stu-id="d1309-119">Query performance is widely variable.</span></span> <span data-ttu-id="d1309-120">例如，有些查詢在短短幾秒鐘內就可以傳回結果，有些查詢則需要花幾分鐘才能傳回。</span><span class="sxs-lookup"><span data-stu-id="d1309-120">For example, some queries return results very quickly, in only a few seconds, while other queries take several minutes to return.</span></span>  
  
-   <span data-ttu-id="d1309-121">彙總資料表不好管理。</span><span class="sxs-lookup"><span data-stu-id="d1309-121">Aggregate tables are difficult to manage.</span></span> <span data-ttu-id="d1309-122">為了改進查詢回應時間， [!INCLUDE[ssSampleDBCoShort](../includes/sssampledbcoshort-md.md)] 的資料倉儲團隊在 **AdventureWorksDW2012** 資料庫中建立幾份彙總資料表。</span><span class="sxs-lookup"><span data-stu-id="d1309-122">In an attempt to improve query response times, the data warehouse team at [!INCLUDE[ssSampleDBCoShort](../includes/sssampledbcoshort-md.md)] built several aggregate tables in the **AdventureWorksDW2012** database.</span></span> <span data-ttu-id="d1309-123">例如，他們建置一個按月彙總銷售量的資料表。</span><span class="sxs-lookup"><span data-stu-id="d1309-123">For example, they built a table that summarizes sales by month.</span></span> <span data-ttu-id="d1309-124">不過，雖然這些彙總資料表可大幅改進查詢效能，但他們經過一段時間建置來維護資料表的基礎架構既脆弱，又容易出錯。</span><span class="sxs-lookup"><span data-stu-id="d1309-124">However, while these aggregate tables greatly improve query performance, the infrastructure that they built to maintain the tables over time is fragile and prone to errors.</span></span>

-   <span data-ttu-id="d1309-128">不同事業單位的使用者對不同資料檢視感興趣。</span><span class="sxs-lookup"><span data-stu-id="d1309-128">Users in different business units are interested in different views of the data.</span></span> <span data-ttu-id="d1309-129">每一個團體對於不相關的資料元素感到困擾及困惑。</span><span class="sxs-lookup"><span data-stu-id="d1309-129">Each group is distracted and confused by data elements that are irrelevant to them.</span></span>  
  
-   <span data-ttu-id="d1309-130">計算邏輯對於需要特殊報表的使用者更是一大挑戰。</span><span class="sxs-lookup"><span data-stu-id="d1309-130">Calculation logic is particularly challenging for users who need specialized reports.</span></span> <span data-ttu-id="d1309-131">因為這類使用者必須個別為每一個報表定義計算邏輯，所以對於計算邏輯如何定義並沒有集中控制的方法。</span><span class="sxs-lookup"><span data-stu-id="d1309-131">Because such users must define the calculation logic separately for each report, there is no centralized control over how the calculation logic is defined.</span></span> <span data-ttu-id="d1309-132">例如，有些使用者知道他們應該使用基本統計技巧，例如移動平均值，但他們不知道如何去建構這樣的計算，而沒有使用這些技巧。</span><span class="sxs-lookup"><span data-stu-id="d1309-132">For example, some users know that they should use basic statistical techniques such as moving averages, but they do not know how to construct such calculations and so do not use these techniques.</span></span>

-   <span data-ttu-id="d1309-138">報表主要重點在於美國的商務資訊。</span><span class="sxs-lookup"><span data-stu-id="d1309-138">Reports are focused primarily on business information in the United States.</span></span> <span data-ttu-id="d1309-139">非美國分公司的使用者很不滿意這項重點，他們想要以不同貨幣和不同語言檢視報表。</span><span class="sxs-lookup"><span data-stu-id="d1309-139">Users in the non-U.S. subsidiaries are very dissatisfied with this focus, and want to be able to view reports in different currencies and different languages.</span></span>  
  
-   <span data-ttu-id="d1309-140">資訊很難稽核。</span><span class="sxs-lookup"><span data-stu-id="d1309-140">Information is difficult to audit.</span></span> <span data-ttu-id="d1309-141">財務部門目前使用的 **AdventureWorksDW2012** 資料庫只做為大量查詢的資料來源。</span><span class="sxs-lookup"><span data-stu-id="d1309-141">The Finance department currently uses the **AdventureWorksDW2012** database only as a source of data from which to query in bulk.</span></span> <span data-ttu-id="d1309-142">他們會將資料下載到個別的試算表，然後花很多時間準備資料及處理試算表。</span><span class="sxs-lookup"><span data-stu-id="d1309-142">They then download the data into individual spreadsheets, and spend significant time preparing the data and manipulating the spreadsheets.</span></span> <span data-ttu-id="d1309-143">因此，公司財務報表很難在整個公司進行準備、稽核和管理的工作。</span><span class="sxs-lookup"><span data-stu-id="d1309-143">The corporate financial reports are therefore difficult to prepare, audit, and manage across the company.</span></span>
