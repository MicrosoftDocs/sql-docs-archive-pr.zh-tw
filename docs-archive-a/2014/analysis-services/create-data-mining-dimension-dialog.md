# <a name="create-data-mining-dimension-dialog"></a>建立資料採礦維度對話
  使用 **[建立資料採礦維度]** 對話方塊，即可依據 OLAP 採礦模型來建立新的資料採礦維度。  
  
## <a name="options"></a>選項  
 **模型名稱**  
 從 [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] 專案裡 OLAP 採礦模型的清單中，選取將作為資料採礦維度基礎的 OLAP 採礦模型。  
  
 **維度名稱**  
 輸入新維度的名稱。  
  
 **建立 Cube**  
 選取即可建立 Cube，該 Cube 會以即將建立的採礦模型維度為基礎。 選取此選項之後，您可以輸入 Cube 的新名稱。
