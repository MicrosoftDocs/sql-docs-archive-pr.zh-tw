# <a name="completing-the-wizard-data-mining-wizard"></a>正在完成精靈 (資料採礦精靈)
  使用 [正在完成精靈]**** 頁面，即可檢閱完成精靈時會建立的採礦結構。 您也可以設定採礦結構的名稱。  
  
 如果選擇建立相關聯的採礦模型，則也可以設定相關聯採礦模型的名稱，然後啟用採礦模型的鑽研功能。  
  
> [!NOTE]  
>  此頁面會依據您在精靈的 [選取定義方法]**** 頁面上，是選取 **[從現有的關聯式資料庫或資料倉儲]** 或 **[從現有的 Cube]** 而變更。  
  
 **如需詳細資訊，請參閱：** [資料採礦精靈 &#40;Analysis Services - 資料採礦&#41;](data-mining/data-mining-wizard-analysis-services-data-mining.md)、[資料採礦設計師](data-mining/data-mining-designer.md)、[建立關聯式採礦結構](data-mining/create-a-relational-mining-structure.md)  
  
## <a name="options"></a>選項  
 **採礦結構名稱**  
 鍵入 [資料採礦精靈]**** 所定義之採礦結構的名稱。  
  
 **採礦模型名稱**  
 鍵入 [資料採礦精靈]**** 所定義之採礦模型的名稱。  
  
 **允許使用鑽研**  
 如果已建立新的採礦模型，請選取支援新採礦模型的鑽研功能。  
  
> [!NOTE]  
>  根據預設，採礦結構的鑽研功能是啟用的。  
  
 如需鑽研選項的詳細資訊，請參閱[鑽研查詢 &#40;資料採礦&#41;](data-mining/drillthrough-queries-data-mining.md)。  
  
 **預覽**  
 顯示要建立的採礦結構。  
  
 **建立採礦模型維度**  
 選取即可依據要建立的採礦模型來建立維度。 選取此選項之後，請輸入要建立之維度的名稱。 選取此選項會啟用 [使用採礦模型維度建立 Cube]****。  
  
> [!NOTE]  
>  如果您在精靈的 [選取定義方法]**** 頁面上選取 [從現有的 Cube]****，則可以使用此選項。  
  
 **使用採礦模型維度建立 Cube**  
 選取即可依據要建立的採礦模型來建立 Cube。 選取此選項之後，請輸入要建立之 Cube 的名稱。  
  
> [!NOTE]  
>  如果您在 [選取定義方法]**** 頁面上選取 [從現有的 Cube]****，並且在精靈的目前頁面上選取 [建立採礦模型維度]****，則可以使用此選項。
