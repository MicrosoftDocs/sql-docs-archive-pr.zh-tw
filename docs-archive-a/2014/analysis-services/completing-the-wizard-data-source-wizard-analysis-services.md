# <a name="completing-the-wizard-data-source-wizard-analysis-services"></a>正在完成精靈 (資料來源精靈) (Analysis Services)
  使用精靈的 **[正在完成精靈]** 頁面，即可檢閱連接字串，並指定新資料來源的名稱。  
  
## <a name="options"></a>選項  
 **資料來源名稱**  
 鍵入名稱，或使用為資料來源提供的預設名稱。  
  
 **預覽**  
 顯示資料來源的連接字串。 按 **[上一步]** ，即可對這些設定進行變更。
